<p>In 1507 the German cartographer Martin Waldseem端ller published a wall map of the world, the <i>Universalis Cosmographia</i>, which was the first to use the name "America". On the same year he also published a set of gores for contructing a globe, also identifying "America". The photos show the process of printing and assembling these gores as a papercraft model, albeit with some irregularities.</p>
